# 1 语法

## 概述

两种的语法

- SCSS（Sassy CSS）是 CSS 的超集，更常用
- Sass 省略括号，使用缩进嵌套，更简洁

## 输入字符集

Dart Sass 目前仅支持 UTF-8 编码。因此，将所有 Sass 样式表编码为 UTF-8 是最安全的。
